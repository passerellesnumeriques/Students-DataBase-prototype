locale.js

locale.load(url) Loads a file of localized strings, at the given url.
To load the common strings, the URL is /locale/common/ (this one is automatically loaded in configuration.js).
To load strings from a component XXX, the URL is /locale/XXX/.
locale.get_string(key,namedValues) Returns the localized string corresponding to the given key.
If the string contains parameters (parts to be replaced), the namedValues should contain the value to be used for each.
