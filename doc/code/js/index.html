The JavaScript part is composed of:
  • The most important is utils.js: it contains functions making the application cross-browser compatible, and provides some useful functions often needed.
  • ajax.js is the recommended way to make AJAX calls, and especially ajax.post_parse_result is recommended to call services on PHP components.
  • common_dialogs.js provides popup windows for errors, questions, information...
  • locale.js is the javascript implementation to provide localized strings
  • Also some UI components like grid, popup window, menu, ... are provided in order to make the application more user-friendly, and uniform among the different pages.
  • field directory, defines editors/displayers for different kind of data: example, if the data is an enumeration of possible values, it will be displayed as a select. This may be useful to create UI components that will automatically adapt to different kind of data to be displayed/edited.
  • animation.js contains useful functions to create animations in the page (like fading out an element).
  • dragndrop.js allows support of drag'n'drop operations
  • drawing.js provides functions to draw on the page, and objects to handle 2D geometry
  • validation.js contains functions to provide visual feedbacks to the user when validating fields of a form
  • Status.js and StatusUI_Top.js provides a way to display status messages (loading, errors, ...)
