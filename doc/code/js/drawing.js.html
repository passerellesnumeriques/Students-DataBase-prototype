Point2D

Class representing a 2D point (x,y).
Constructor
Point2D(x,y) initialize the object with the given coordinates (x, y).
Attributes
x
y 		Horizontal (x) and vertical (y) coordinate of the point.

Line2D

Class representing a line between two 2D points.
Constructor
Line2D(point1, point2) point1 and point2 are Point2D objects.
Attributes
pt1
pt2 		The 2 points defining the line.
Methods
getEquation() Returns an array of 3 floating points [A,B,C] corresponding to the 2D linear equation.
getDistanceSquared(pt) Returns the distance squared between this line and the given point pt.
getDistance(pt) Returns the distance between this line and the given point pt.
Equivalent to Math.sqrt(getDistanceSquared(pt)).

drawing

The object drawing provides the following functions:
horizontal_connector(e1,e2) Draws a line that will connect the 2 given elements e1 and e2 horizontally.
That means the line will be between the right edge of e1 and the left edge of e2.
