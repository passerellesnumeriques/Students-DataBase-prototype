A collapsable section, is a block having a title and a content. By clicking on the title, the user can hide or show the content.
Example:
The title of the section
The content of the section.
This will be hidden by clicking on the title

Constructor
collapsable_section(element) Creates a collapsable section on the given element.
element can be either the element itself, or its ID.
It expects to find two children: one having the class collapsable_section_header, which contains the title, and one having the class collapsable_section_content, which is the content.
Methods
toggle() Hide/Show the content.
