validation.js

validation_error(input, message) This will highligh in red the given input, and will display the error message in the element having as ID the ID or the name of the input followed by "_validation" if it exists.
For example, if we have >input name='username' ... <, it will looks for an element having ID username_validation to put the message.
If the element is found, the message is put inside, and its CSS style visibility is set to visible.
Example:
validation_ok(input) Similar to validation_error but in case the validation is correct: remove the red, and hide the element containing the error message.
